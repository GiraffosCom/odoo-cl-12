Autores

Resumen

Permite seleccionar partidas abiertas por pagar según varios criterios de búsqueda. (Diario Contable, Fecha de Vencimiento, Vía de Pago, etc.) Tras realiza el pago se genera el archivo a enviar al banco. Se contabiliza el pago una vez confirmada la carga y aceptación del pago por parte del banco.
